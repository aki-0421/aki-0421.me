## どのようなアプリか

線画の画像を入力すると、AIが下塗りをしたPSDを作成するWebアプリ

## 作ろうと思った理由

2022年に画像生成AIが普及したものの、製作の過程で直接役に立つ技術ではなかった。

そのため、イラスト製作の工程の中で自動化できそうな工程をAIにまかせる方法を考えてみたかった。

原案: <Link href="https://github.com/mattyamonaca/auto_undercoat">mattyamonaca/auto_undercoat</Link>

## 構成

- **フロントエンド**
    - Webアプリ
        - Next.js(App Router)
        - Chakra UI
            - デフォルトのUIが美しい
            - テーマを作成できるためコードを簡潔にできる
        - Framer Motion
            - コンポーネントにアニメーションを追加するため
- **バックエンド**
    - APIサーバー
        - Golang
    - AIでの画像処理用サーバー
        - Python
- **サーバー**
    - Google Cloud Platform
        - Cloud Run
            - WebアプリとAPIサーバーを動かすために利用した
    - Runpod.io
        - AIでの処理にGPUサーバーが必要なため
        - 利用した時間のみ1秒単位で課金のため使いやすい
